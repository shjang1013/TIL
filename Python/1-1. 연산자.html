# 연산자


# x의 y제곱 ** 연산자

** 연산자는 x의 y제곱 값을 돌려주는 연산자다.

# ex) ** 연산자 예제

>>> a = 3
>>> b = 4
>>> a ** b
81

# 나머지 연산자 % 연산자

% 연산자는 나눗셈의 나머지 값을 돌려주는 연산자다.

# ex) % 연산자 예제

>>> 7 % 3
1
>>> 3 & 7
3

# 몫을 반환하는 // 연산자

/ 연산자는 나눈 값을 반환하지만, // 연산자는 나눗셈 후 몫을 반환한다.

# ex) // 연산자 예제

>>> 7 // 4
1
>>> 7 / 4
1.75
